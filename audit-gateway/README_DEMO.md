```bash

# 等待 ClickHouse 就绪（约15秒）
```


## 常见问题

### Q: 生成的数据量太大/太小怎么办？

编辑 `generate_demo.py`，修改时间粒度：

```python
# 当前是每5分钟一个数据点
# 想要更少数据，改成 10 分钟
# 想要更多数据，改成 1 分钟
current += timedelta(minutes=5)  # 改这里
```

### Q: 想要更多敏感词检测？

修改 `RISK_LEVELS` 权重：

```python
RISK_LEVELS = [("low", 0.50), ("medium", 0.35), ("high", 0.15)]  # 增加高风险比例
```

### Q: 想要特定部门更多数据？

修改 `DEPARTMENTS` 权重：

```python
DEPARTMENTS = [
    ("研发部", 0.50),  # 增加研发部比例
    ("产品部", 0.15),
    ...
]
```
